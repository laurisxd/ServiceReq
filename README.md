This is basic Service ticketing project, by using Java Spring Boot.

To access the system, first of all open your IDE and import the project.
After importing, run it.

After that you need to setup the database. I used the MySQL Workbench.
Make a new database with username - root, and password - root.

Also if you want to use any other credentials, 
please change it in code, in src/main/resources/application.properties.

After that, make a two tables.
ticket and user.
